Sensores ambientais

Fusão inteligente de dados de satélite e sensores de superfície in-situ para ajudar a compreender o nosso planeta em mudança

  • Projeto
  • Maio 2021 - Presente
  • Python
Descripción

A fim de avançar na compreensão da mudança ambiental e melhorar a previsão, temos de encontrar formas inteligentes de combinar dados de satélite, sensores de campo e resultados de simuladores ambientais que tenham em conta os princípios físicos (por exemplo, modelos climáticos). Para reconciliar as escalas espaciais e as diversas modalidades destes dados, este projecto visa construir um conjunto de ferramentas impulsionado por estudos de casos do mundo real. Os resultados contribuem para a construção de mais capacidade de investigação no Reino Unido no desenvolvimento de Gémeos Digitais do ambiente natural.

O projecto consiste em três linhas de trabalho:

  • Linha 1: Aprendizagem profunda para a instalação inteligente de sensores ambientais.
  • Linha 2: Detecção e monitorização de alterações ambientais utilizando a teledetecção.
  • Linha 3: Criar princípios para os gêmeos digitais do ambiente natural.

    • Como parte do meu postdoc sobre o projecto, o meu papel é transversal em cada uma das vertentes. Em particular, as minhas principais contribuições estão na linha 1 onde, em colaboração com outros investigadores do British Antarctic Survey e Met Office, estamos a desenvolver uma ferramenta generalizável baseada numa modelação probabilística capaz de combinar dados de diferentes modalidades (dados em grelha, dados históricos in situ e dados auxiliares) para ter observações mais finas e mais locais das variáveis ambientais. Além disso, com algoritmos greedy, a ferramenta é proposta para sugerir onde instalar sensores locais a fim de reduzir a incerteza da variável ambiental modelada. Dentro dos estudos de caso propostos, a minha variável ambiental a ser modelada é a humidade do solo. Os dados de entrada são ERA5, as observações 2013-2019 das estações COSMOS-UK e o modelo de terreno digital multi-resolução de 1 km (GMTED2010).

    Finalmente, motivado pela diversidade de projetos e investigadores com os quais colaboro no projecto, bem como comunidades ativas no desenvolvimento de software aberto, tais como Pangeo , desenvolvi também a comunidade de Ciência de Dados Ambientais. Para este fim, estou a manter um recurso desenvolvido no Jupyter Book que hospeda conteúdo executável em formato de cadernos Jupyter, bem como registos de reuniões mensais da comunidade em que discutimos tópicos relacionados com a ciência dos dados ambientais.
